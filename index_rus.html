Мои главные проекты

  • Dota WebSite
    Django + DRF + Docker

    Dota WebSite - образовательный проект, написанный на фрэймворке python - Django.
    Моей целью было изучить основы Django и другие полезные инструменты, такие как Django REST Framework, Docker, приципы of CI/CD и т.д.

    Aiogram + Asyncio + Aiohttp

    BS4 + Tkinter + Requests

    Это приложение использует данные центрального банка России для обмена одной валюты на другую.
